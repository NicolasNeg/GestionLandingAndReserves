# Map assets (parque acuático)

Coloca aquí los recursos referenciados por el editor / vista pública Konva.

| Ruta sugerida | Uso |
|---------------|-----|
| `base/waterpark-map.webp` | Imagen base ilustrada del parque (fondo del lienzo) |
| `sprites/*.webp` | Sprites transparentes (palapa, mesa, etc.) |
| `icons/*.svg` | Iconos para marcadores y filtros |
| `textures/*.webp` | Texturas opcionales (agua, arena) |
| `patterns/restricted-hatch.svg` | Patron de zona restringida |

Si un archivo no existe, el editor usa degradados / colores por defecto. Reemplaza los archivos sin cambiar rutas en la configuración del mapa (`background.url` en el JSON).
